# flappy-bird-react

very simple and naive flappy bird implementation in react-js.

- git clone.
- yarn install.
- yarn start.


have fun :)

[Play a live demo - click here](https://flappy-bird-react-friedhelmensch.vercel.app/) - Press "s" to make the "bird" jump
